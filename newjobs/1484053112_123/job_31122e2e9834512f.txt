Data Scientist

Submit to Job(s)    
Req ID :
377370BR
Job Code/Title :
E1624:Manufacturing Engineer Stf
Job Description :
This position will be responsible for the analysis of vast quantities of
machine data collected from fleets of complex machines such as engines and
helicopters.  The applicant is expected to collaborate with other data
scientists, engineers, software teams, management, and other stakeholders to
conduct data wrangling / mining, perform exploratory analysis, use machine
learning to build models, generate visualizations, and create & present
reports. The applicant should have a BS or MS in Engineering, Data Science,
Computer Science, Mathematics or Statistics, and be comfortable writing code
for data analysis (preferably with Python, Scala, R, or Java).Required skills
include: Basic theoretical foundation in statistics and machine learningGood
problem solving and organizational skillsExcellent written and verbal
communication skills (i.e. the ability to communicate complex analysis with
clarity and precision) Familiarity with standard applied machine learning and
data mining techniquesPassionate about empirical research and naturally
inquisitiveExperience in data analysis with Python, Scala, R, and/or Java
 Desired skills include: Comfort manipulating and analyzing complex,
high-volume, high-dimensionality data from varying sourcesBackground in general
engineering fields such as mechanical or electrical engineeringExperience with
large and/or complex data setsFamiliarity with Hadoop / MapReduce, Pig, Hive,
and SparkFamiliarity with relational databases and SQLFamiliarity with
AWSExperience with Linux and complex computer networks
