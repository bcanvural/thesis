                        Intern - Software Engineer for Data Analytics/Machine Learning
                         

                        Description:

                        The candidate is required to have hands on experience in software development
                        projects involving design, implementation and testing. The ideal candidate
                        would have strong software development skills using tools such as R, Python and
                        C/C++. Being familiar with or have background in telecommunication system and/
                        or statistical & machine learning algorithms would be a big plus. This position
                        is based out of Bridgewater, NJ.


                        Responsibilities:
                         
                        1. Machine learning algorithm implementation with emphasis in correctness and
                        efficiency. The problems being addressed will be related telecommunication
                        system monitoring, modeling/prediction, and optimization.
                        2. Prototyping and validation using R, Python, some C++ and statistical and
                        data mining tools.

                        Requirements
                        1. Software development experience using modern general purpose languages such
                        as Python and C++ and Statistical/data mining tools/languages such as R and
                        Weka.
                        2. A plus to have knowledge of statistical/machine learning techniques such as
                        feature extraction, clustering, linear regression, logistic regression,
                        forecasting, hypothesis testing, etc.
                        3. A plus to have background knowledge and/or experience in telecommunication
                        system (3G/4G Mobile radio access networks/core, OAM, KPI etc), service
                        provider or a 3rd party analytics company.
                        5. Excellent verbal and written communication skill; ability to work well in a
                        team environment.
                        6. MS or PhD in Computer Science or EE.
